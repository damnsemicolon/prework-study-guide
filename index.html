How to Pull Request a Feature Branch into the Main Branch


  1. Type in Git "git add -A"; this commit all files in the working branch
  2. Type in Git "git commit -m 'add short descriptive message as commit'"
  3. Type in Git "git pull origin main"
  4. Type in Git git push origin 'branch-name'"
  5. Navigate to 'branch-name' in Github, click the "pull requst" tab, then the green "New pull request"
  6. Give your PR a title and short description, then the green "Create pull request"
  7. Finally, click the "Merge pull request" button
